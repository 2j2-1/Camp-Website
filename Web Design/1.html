Tags

Tags are how you write text in html - this is the basics of all websites. A tag's basic job is to tell the internet browser how the text should be shown. A tag consits of 2 part an intial <h1> with text in the middle followed by </h1> eg <h1>Hello World</h1> would diplay

<title>

Inbetween the tags is the title of your website, this is the only thing that should be modified within the head tags at this time

<body>

Neraly everything you write in html will be between the body tags

The First Tag You Will Write Is The Header Tag, <h1>

This tag makes the text inside a header. Using different numbers after the h will result in smaller text but still larger then the main text.

The Next Tag Is The Paragraph Tag, <p>

This tag is used for the main body of the page.
